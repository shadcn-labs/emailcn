Authentication emails are used for user verification and account security.

## Variants

### OTP Default

One-time password email with verification code.

```tsx title="registry/blocks/auth-otp-default.tsx"
// Subject: Your verification code for {_productName}
// ...
```

### OTP Twitch

OTP styled like Twitch's verification emails.

```tsx title="registry/blocks/auth-otp-twitch.tsx"
// ...
```

### Magic Link Default

Simple magic link for passwordless login.

```tsx title="registry/blocks/auth-magic-link-default.tsx"
// ...
```

### Magic Link Raycast

Magic link styled like Raycast's emails.

```tsx title="registry/blocks/auth-magic-link-raycast.tsx"
// ...
```

### Magic Link Stripe

Magic link styled like Stripe's authentication emails.

```tsx title="registry/blocks/auth-magic-link-stripe.tsx"
// ...
```

### Password Reset Default

Password reset instructions.

```tsx title="registry/blocks/auth-password-reset-default.tsx"

// ...
```

## Props

| Prop               | Type     | Default             | Description             |
| ------------------ | -------- | ------------------- | ----------------------- |
| `logoUrl`          | `string` | -                   | URL of the product logo |
| `_logoAlt`         | `string` | `"Logo"`            | Alt text for logo       |
| `otpCode`          | `string` | `"123456"`          | OTP code                |
| `expiresInMinutes` | `number` | `10`                | Code expiration time    |
| `magicLink`        | `string` | `"#"`               | Magic link URL          |
| `_recipientEmail`  | `string` | `"you@example.com"` | Recipient email         |
| `_productName`     | `string` | `"Acme"`            | Product name            |
