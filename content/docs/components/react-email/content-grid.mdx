## API Reference

### Content Grid

| Prop      | Type         | Default        | Description         |
| --------- | ------------ | -------------- | ------------------- |
| `theme`   | `EmailTheme` | `defaultTheme` | Email theme styling |
| `columns` | `number`     | `2`            | Number of columns   |
| `gap`     | `string`     | -              | Gap between items   |
