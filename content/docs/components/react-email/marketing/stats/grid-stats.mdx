
```tsx

```

## API Reference

| Prop      | Type                                             | Default        | Description         |
| --------- | ------------------------------------------------ | -------------- | ------------------- |
| `tone`    | `"boxed" \| "outlined" \| "accent" \| "border"`  | `"boxed"`      | Grid cell style     |
| `variant` | `"default" \| "slanted-left" \| "slanted-right"` | `"default"`    | Section tilt        |
| `theme`   | `TailwindConfig`                                 | `defaultTheme` | Email theme styling |
