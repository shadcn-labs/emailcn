## API Reference

| Prop                      | Type                       | Default         | Description                                     |
| ------------------------- | -------------------------- | --------------- | ----------------------------------------------- |
| `theme`                   | `TailwindConfig`           | `defaultTheme`  | React Email Tailwind configuration              |
| `variant`                 | `CTAWithImageStripVariant` | `"boxed-right"` | Selects the exact side and strip treatment      |
| `heading` / `subtext`     | `string`                   | copy            | CTA heading and supporting text                 |
| `ctaLabel` / `ctaHref`    | `string`                   | CTA             | Primary action label and destination            |
| `imageSrc1` … `imageSrc5` | `string`                   | Variant images  | Image URLs used by the strip                    |
| `imageAlt1` … `imageAlt5` | `string`                   | `""`            | Alternative text for each strip image           |
| Color props               | `string`                   | colors          | Canvas, surface, strip, text, and button colors |
