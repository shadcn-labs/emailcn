## API Reference

| Prop                                                                                         | Type                               | Default             | Description                            |
| -------------------------------------------------------------------------------------------- | ---------------------------------- | ------------------- | -------------------------------------- |
| `theme`                                                                                      | `TailwindConfig`                   | `defaultTheme`      | React Email Tailwind configuration     |
| `variant`                                                                                    | `CTAWithTitleAndActionLeadVariant` | `"title-and-lead"`  | Selects the exact Mailviews CTA layout |
| `heading` / `subtext`                                                                        | `string`                           | Mailviews copy      | Lead heading and supporting text       |
| `ctaLabel` / `ctaHref`                                                                       | `string`                           | Variant CTA         | Primary button label and destination   |
| `secondaryCtaLabel` / `secondaryCtaHref`                                                     | `string`                           | Mailviews secondary | Optional secondary action              |
| `signoff`                                                                                    | `string`                           | Mailviews signoff   | Copy beneath the minimal CTA           |
| `pageBackgroundColor` / `backgroundColor`                                                    | `string`                           | Mailviews colors    | Canvas and CTA surface colors          |
| `headingColor` / `textColor`                                                                 | `string`                           | Mailviews colors    | Heading and supporting text colors     |
| `primaryButtonBackgroundColor` / `primaryButtonTextColor`                                    | `string`                           | Mailviews colors    | Primary button colors                  |
| `secondaryButtonBackgroundColor` / `secondaryButtonTextColor` / `secondaryButtonBorderColor` | `string`                           | Mailviews colors    | Secondary button colors and border     |
