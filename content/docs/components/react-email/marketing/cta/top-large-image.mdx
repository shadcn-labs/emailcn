## API Reference

| Prop                    | Type             | Default        | Description                              |
| ----------------------- | ---------------- | -------------- | ---------------------------------------- |
| `theme`                 | `TailwindConfig` | `defaultTheme` | React Email Tailwind configuration       |
| `heading` / `subtext`   | `string`         | copy           | CTA heading and supporting text          |
| `ctaLabel` / `ctaHref`  | `string`         | CTA            | Primary action label and destination     |
| `imageSrc` / `imageAlt` | `string`         | image          | Leading image URL and alternative text   |
| Color props             | `string`         | colors         | Canvas, surface, text, and button colors |
