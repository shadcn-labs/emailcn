## API Reference

| Prop                   | Type                                 | Default          | Description                                  |
| ---------------------- | ------------------------------------ | ---------------- | -------------------------------------------- |
| `theme`                | `TailwindConfig`                     | `defaultTheme`   | React Email Tailwind configuration           |
| `variant`              | `BoxedCTAWithBackgroundImageVariant` | `"flush-light"`  | Selects the exact Mailviews spacing and tone |
| `heading` / `subtext`  | `string`                             | Mailviews copy   | CTA heading and supporting text              |
| `ctaLabel` / `ctaHref` | `string`                             | Mailviews CTA    | Primary action label and destination         |
| `backgroundImageSrc`   | `string`                             | Mailviews glow   | Background image behind the CTA card         |
| Color props            | `string`                             | Mailviews colors | Canvas, card, text, and button colors        |
