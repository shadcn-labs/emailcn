## API Reference

| Prop                   | Type                                 | Default         | Description                           |
| ---------------------- | ------------------------------------ | --------------- | ------------------------------------- |
| `theme`                | `TailwindConfig`                     | `defaultTheme`  | React Email Tailwind configuration    |
| `variant`              | `BoxedCTAWithBackgroundImageVariant` | `"flush-light"` | Selects the exact spacing and tone    |
| `heading` / `subtext`  | `string`                             | copy            | CTA heading and supporting text       |
| `ctaLabel` / `ctaHref` | `string`                             | CTA             | Primary action label and destination  |
| `backgroundImageSrc`   | `string`                             | glow            | Background image behind the CTA card  |
| Color props            | `string`                             | colors          | Canvas, card, text, and button colors |
