## API Reference

| Prop                                     | Type                                            | Default           | Description                                             |
| ---------------------------------------- | ----------------------------------------------- | ----------------- | ------------------------------------------------------- |
| `theme`                                  | `TailwindConfig`                                | `defaultTheme`    | React Email Tailwind configuration                      |
| `variant`                                | `"default" \| "avatars-top" \| "single-avatar"` | `"default"`       | Selects the exact Mailviews layout and copy             |
| `heading` / `subtext`                    | `string`                                        | Variant copy      | CTA heading and supporting text                         |
| `ctaLabel` / `ctaHref`                   | `string`                                        | Variant CTA       | Primary action label and destination                    |
| `secondaryCtaLabel` / `secondaryCtaHref` | `string`                                        | Single-avatar CTA | Optional secondary action                               |
| Avatar props                             | `string`                                        | Mailviews avatars | Four avatar URLs and alternative-text values            |
| Color props                              | `string`                                        | Mailviews colors  | Canvas, surface, avatar border, text, and button colors |
