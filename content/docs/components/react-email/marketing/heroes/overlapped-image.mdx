## API Reference

| Prop                                             | Type                                             | Default          | Description                                        |
| ------------------------------------------------ | ------------------------------------------------ | ---------------- | -------------------------------------------------- |
| `theme`                                          | `TailwindConfig`                                 | `defaultTheme`   | React Email Tailwind configuration                 |
| `variant`                                        | `"default" \| "slanted-left" \| "slanted-right"` | `"default"`      | Selects the exact Mailviews layout and content set |
| `eyebrow` / `heading` / `subheading`             | `string`                                         | Variant copy     | Product title copy                                 |
| `description`                                    | `string`                                         | Variant copy     | Supporting product description                     |
| `ctaLabel` / `ctaHref`                           | `string`                                         | Mailviews CTA    | Button label and destination                       |
| `backgroundImageSrc` / `imageSrc` / `imageAlt`   | `string`                                         | Variant imagery  | Background and overlapping product imagery         |
| `logoSrc` / `logoAlt` / `logoHref`               | `string`                                         | Light logo       | Brand logo and destination                         |
| `pageBackgroundColor` / `contentBackgroundColor` | `string`                                         | Mailviews colors | Canvas and content surface colors                  |
| `textColor` / `mutedTextColor`                   | `string`                                         | Variant colors   | Primary and supporting copy colors                 |
| `buttonBackgroundColor` / `buttonTextColor`      | `string`                                         | Mailviews colors | CTA colors                                         |
