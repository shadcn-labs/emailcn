## API Reference

| Prop                    | Type                                | Default           | Description                                      |
| ----------------------- | ----------------------------------- | ----------------- | ------------------------------------------------ |
| `theme`                 | `TailwindConfig`                    | `defaultTheme`    | React Email Tailwind configuration               |
| `variant`               | `"overlay-left" \| "overlay-right"` | `"overlay-left"`  | Places the translucent content panel on one side |
| `eyebrow`               | `string`                            | `"Brew Stories"`  | Small label above the heading                    |
| `heading`               | `string`                            | Mailviews copy    | Main article heading                             |
| `subheading`            | `string`                            | `"January…"`      | Date or supporting line                          |
| `description`           | `string`                            | Mailviews copy    | Supporting paragraph                             |
| `ctaLabel` / `ctaHref`  | `string`                            | Mailviews CTA     | Button label and destination                     |
| `imageSrc` / `imageAlt` | `string`                            | Mailviews imagery | Full-width background image and label            |
| `logoSrc` / `logoAlt`   | `string`                            | Light logo        | Logo shown at the top of the overlay             |
| `logoHref`              | `string`                            | Example URL       | Logo destination                                 |
| `pageBackgroundColor`   | `string`                            | `"#f1f5f9"`       | Email-canvas background                          |
| `overlayColor`          | `string`                            | Dark 80% overlay  | Content-panel background                         |
| `textColor`             | `string`                            | `"#f9fafb"`       | Overlay text color                               |
| `buttonBackgroundColor` | `string`                            | `"#4f46e5"`       | CTA background                                   |
| `buttonTextColor`       | `string`                            | `"#fffffe"`       | CTA text color                                   |
