## API Reference

| Prop                                                       | Type                        | Default           | Description                                       |
| ---------------------------------------------------------- | --------------------------- | ----------------- | ------------------------------------------------- |
| `theme`                                                    | `TailwindConfig`            | `defaultTheme`    | React Email Tailwind configuration                |
| `variant`                                                  | `HeroBlockWithBleedVariant` | `"left-centered"` | Selects side and vertical placement               |
| `eyebrow` / `heading` / `subheading`                       | `string`                    | Mailviews copy    | Main conservation copy                            |
| `description`                                              | `string`                    | Mailviews copy    | Supporting paragraph                              |
| `ctaLabel` / `ctaHref`                                     | `string`                    | Mailviews CTA     | Button label and destination                      |
| `backgroundImageSrc` / `imageAlt`                          | `string`                    | Landscape imagery | Full-bleed background and accessible label        |
| `logoSrc` / `logoAlt` / `logoHref`                         | `string`                    | Light logo        | Brand logo used by centered and bottom placements |
| `pageBackgroundColor` / `backgroundColor` / `overlayColor` | `string`                    | Mailviews colors  | Canvas, fallback, and translucent block colors    |
| `textColor`                                                | `string`                    | `"#f9fafb"`       | Hero copy color                                   |
| `buttonBackgroundColor` / `buttonTextColor`                | `string`                    | Mailviews colors  | CTA colors                                        |
