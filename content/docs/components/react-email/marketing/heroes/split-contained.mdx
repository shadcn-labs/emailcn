## API Reference

| Prop                                        | Type                                                                                                                                                                                               | Default               | Description                                               |
| ------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------- | --------------------------------------------------------- |
| `theme`                                     | `TailwindConfig`                                                                                                                                                                                   | `defaultTheme`        | React Email Tailwind configuration                        |
| `variant`                                   | `"single-image-left" \| "single-image-right" \| "square-images-left" \| "square-images-right" \| "portrait-top-left" \| "portrait-top-right" \| "portrait-bottom-left" \| "portrait-bottom-right"` | `"single-image-left"` | Selects one of the eight Mailviews layouts and its preset |
| `eyebrow` / `heading` / `subheading`        | `string`                                                                                                                                                                                           | Variant preset        | Hero title content                                        |
| `description` / `price`                     | `string`                                                                                                                                                                                           | Variant preset        | Supporting copy and optional bold price                   |
| `ctaLabel` / `ctaHref`                      | `string`                                                                                                                                                                                           | Variant preset        | Button label and destination                              |
| `primaryImageSrc` / `primaryImageAlt`       | `string`                                                                                                                                                                                           | Mailviews imagery     | Single background or first stacked image                  |
| `secondaryImageSrc` / `secondaryImageAlt`   | `string`                                                                                                                                                                                           | Mailviews imagery     | Second stacked image; unused by single-image variants     |
| `logoSrc` / `logoAlt`                       | `string`                                                                                                                                                                                           | Variant preset        | Light or dark Mailviews logo                              |
| `pageBackgroundColor` / `backgroundColor`   | `string`                                                                                                                                                                                           | Variant preset        | Email canvas and hero surface                             |
| `textColor` / `mutedTextColor`              | `string`                                                                                                                                                                                           | Variant preset        | Primary and supporting copy colors                        |
| `buttonBackgroundColor` / `buttonTextColor` | `string`                                                                                                                                                                                           | Mailviews colors      | CTA colors                                                |
