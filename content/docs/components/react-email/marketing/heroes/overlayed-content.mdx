## API Reference

| Prop                                        | Type                                             | Default        | Description                                       |
| ------------------------------------------- | ------------------------------------------------ | -------------- | ------------------------------------------------- |
| `theme`                                     | `TailwindConfig`                                 | `defaultTheme` | React Email Tailwind configuration                |
| `variant`                                   | `"default" \| "slanted-left" \| "slanted-right"` | `"default"`    | Selects the flat or slanted ending                |
| `eyebrow`                                   | `string`                                         | `"Fresh Drop"` | Accent label above the heading                    |
| `headingAccent` / `heading`                 | `string`                                         | copy           | Bold first word and light remaining product title |
| `subheading`                                | `string`                                         | copy           | Supporting availability line                      |
| `description`                               | `string`                                         | copy           | Centered product description                      |
| `ctaLabel` / `ctaHref`                      | `string`                                         | CTA            | Button label and destination                      |
| `imageSrc` / `imageAlt`                     | `string`                                         | imagery        | Full hero background and label                    |
| `logoSrc` / `logoAlt` / `logoHref`          | `string`                                         | Mono insignia  | Centered brand mark and destination               |
| `pageBackgroundColor`                       | `string`                                         | `"#f1f5f9"`    | Email-canvas background                           |
| `slantColor`                                | `string`                                         | `"#fffffe"`    | Triangular lower-edge color                       |
| `textColor` / `accentColor`                 | `string`                                         | colors         | Main and accent text colors                       |
| `buttonBackgroundColor` / `buttonTextColor` | `string`                                         | colors         | CTA colors                                        |
