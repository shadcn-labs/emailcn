## Usage

```tsx

```

```tsx

  {...FooterWith2ColumnMenuAndDivider.PreviewProps}
  variant="left-logo"
/>
```

## API Reference

| Prop      | Type                          | Default       | Description                         |
| --------- | ----------------------------- | ------------- | ----------------------------------- |
| `variant` | `"left-logo" \| "right-logo"` | `"left-logo"` | Selects the Mailviews logo position |
| `theme`   | `EmailTheme`                  | default theme | Email theme styling                 |
