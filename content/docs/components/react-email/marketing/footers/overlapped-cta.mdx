## Usage

```tsx
import { FooterWithOverlappedCta } from "@/components/emails/marketing/footers/footer-with-overlapped-cta";
```

```tsx
<FooterWithOverlappedCta
  {...FooterWithOverlappedCta.PreviewProps}
  variant="content"
  logoPosition="left"
/>
```

## API Reference

| Prop           | Type                             | Default       | Description                        |
| -------------- | -------------------------------- | ------------- | ---------------------------------- |
| `variant`      | `FooterWithOverlappedCtaVariant` | `"content"`   | Selects the lower Mailviews layout |
| `logoPosition` | `"left" \| "right"`              | `"left"`      | Mirrors supported layouts          |
| `theme`        | `EmailTheme`                     | default theme | Email theme styling                |
