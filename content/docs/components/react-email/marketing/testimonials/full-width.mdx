## API Reference

| Prop      | Type                                | Default        | Description         |
| --------- | ----------------------------------- | -------------- | ------------------- |
| `variant` | `"default" \| "overlapping-avatar"` | `"default"`    | Avatar arrangement  |
| `quote`   | `string`                            | copy           | Testimonial copy    |
| `theme`   | `TailwindConfig`                    | `defaultTheme` | Email theme styling |
