## API Reference

| Prop      | Type                               | Default        | Description              |
| --------- | ---------------------------------- | -------------- | ------------------------ |
| `variant` | `"default" \| "boxed" \| "accent"` | `"default"`    | Mailviews layout variant |
| `theme`   | `TailwindConfig`                   | `defaultTheme` | Email theme styling      |
