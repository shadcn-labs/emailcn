## Usage

```tsx

<ThreeColumnsTeamGrid variant="default" />;
```

## API Reference

| Prop      | Type                                                      | Default        | Description              |
| --------- | --------------------------------------------------------- | -------------- | ------------------------ |
| `variant` | `"default" \| "with-accent" \| "bordered" \| "with-hero"` | `"default"`    | Mailviews layout variant |
| `theme`   | `TailwindConfig`                                          | `defaultTheme` | Email theme styling      |
