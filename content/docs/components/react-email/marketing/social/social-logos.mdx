description: "Social logo tiles with all eight Mailviews layout variants"

`SocialLogos` combines the eight Mailviews tile layouts because they share the
same content and differ only in framing, fill, and border radius.

## Usage

```tsx
import { SocialLogos } from "@/components/emails/marketing/social/social-logos";

<SocialLogos variant="outlined-circle-tiles" />;
```

## API Reference

| Prop          | Type                                                                                                                                                             | Default             | Description                  |
| ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------- | ---------------------------- |
| `variant`     | `"square-tiles" \| "squared-box" \| "circle-tiles" \| "pill-box" \| "outlined-square-tiles" \| "outlined-circle-tiles" \| "outlined-box" \| "outlined-pill-box"` | `"square-tiles"`    | Mailviews tile layout        |
| `items`       | `SocialLogoItem[]`                                                                                                                                               | Mailviews icons     | Social links and icon assets |
| `title`       | `string`                                                                                                                                                         | `"Connect with us"` | Section title                |
| `description` | `string`                                                                                                                                                         | Mailviews copy      | Supporting text              |
| `theme`       | `TailwindConfig`                                                                                                                                                 | `defaultTheme`      | Email theme configuration    |
