## API Reference

| Prop          | Type                     | Default             | Description                  |
| ------------- | ------------------------ | ------------------- | ---------------------------- |
| `items`       | `SimpleSocialLogoItem[]` | Mailviews icons     | Social links and icon assets |
| `title`       | `string`                 | `"Connect with us"` | Section title                |
| `description` | `string`                 | Mailviews copy      | Supporting text              |
| `theme`       | `TailwindConfig`         | `defaultTheme`      | Email theme configuration    |
