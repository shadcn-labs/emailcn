## API Reference

| Prop                      | Type             | Default         | Description                                     |
| ------------------------- | ---------------- | --------------- | ----------------------------------------------- |
| `theme`                   | `TailwindConfig` | `defaultTheme`  | Renderer theme configuration.                   |
| `heading`                 | `string`         | Layout-specific | Optional section heading where supported.       |
| `imageSrc*` / `imageAlt*` | `string`         | Example imagery | Post image sources and labels.                  |
| `title*` / `excerpt*`     | `string`         | Example copy    | Post titles and supporting excerpts.            |
| Metadata props            | `string`         | Layout-specific | Author, host, episode, badge, or date metadata. |
