## Usage

```tsx
import { BasicLogoCloud } from "@/components/emails/marketing/logos/basic-logo-cloud";
```

```tsx
<BasicLogoCloud {...BasicLogoCloud.PreviewProps} variant="with-title" />
```

## API Reference

| Prop      | Type                    | Default        | Description                         |
| --------- | ----------------------- | -------------- | ----------------------------------- |
| `variant` | `BasicLogoCloudVariant` | `"full"`       | Selects the Mailviews content state |
| `theme`   | Renderer theme          | `defaultTheme` | Renderer theme configuration        |
