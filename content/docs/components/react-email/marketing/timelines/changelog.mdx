## API Reference

| Prop      | Type                                                                                                | Default        | Description                  |
| --------- | --------------------------------------------------------------------------------------------------- | -------------- | ---------------------------- |
| `variant` | `"muted-left" \| "muted-right" \| "basic-left" \| "basic-right" \| "accent-left" \| "accent-right"` | `"muted-left"` | Timeline state and alignment |
| `theme`   | `TailwindConfig`                                                                                    | `defaultTheme` | Email theme styling          |
