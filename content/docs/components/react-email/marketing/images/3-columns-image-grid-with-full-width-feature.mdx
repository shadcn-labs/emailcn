## API Reference

| Prop                | Type                                               | Default            | Description                                     |
| ------------------- | -------------------------------------------------- | ------------------ | ----------------------------------------------- |
| `theme`             | Renderer theme                                     | `defaultTheme`     | Renderer theme configuration                    |
| `variant`           | `ThreeColumnsImageGridWithFullWidthFeatureVariant` | `"full-width-top"` | Selects feature order and overlay mode          |
| Feature image props | `string`                                           | data               | Full-width feature image, link, and copy        |
| Grid image props    | `string`                                           | data               | Three images, links, alternative text, and copy |
| Color props         | `string`                                           | colors             | Canvas, surface, and overlay text colors        |
