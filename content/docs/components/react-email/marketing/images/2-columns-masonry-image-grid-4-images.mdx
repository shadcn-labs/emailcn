## API Reference

| Prop                  | Type                                           | Default          | Description                                   |
| --------------------- | ---------------------------------------------- | ---------------- | --------------------------------------------- |
| `theme`               | `TailwindConfig`                               | `defaultTheme`   | React Email Tailwind configuration            |
| `variant`             | `TwoColumnsMasonryImageGridWith4ImagesVariant` | `"stacked-left"` | Selects side, feature order, and overlay mode |
| Feature image props   | `string`                                       | data             | Full-width feature image, link, and copy      |
| Landscape image props | `string`                                       | data             | Two stacked landscape images, links, and copy |
| Portrait image props  | `string`                                       | data             | Portrait image, link, and copy                |
| Color props           | `string`                                       | colors           | Canvas, surface, and overlay text colors      |
