## API Reference

| Prop                        | Type                         | Default           | Description                               |
| --------------------------- | ---------------------------- | ----------------- | ----------------------------------------- |
| `theme`                     | `TailwindConfig`             | `defaultTheme`    | React Email Tailwind configuration        |
| `variant`                   | `TwoColumnsImageGridVariant` | `"square-images"` | Selects image ratio and overlay treatment |
| `imageSrc1` / `imageSrc2`   | `string`                     | images            | Grid image URLs                           |
| `imageAlt1` / `imageAlt2`   | `string`                     | `""`              | Alternative text for plain-image variants |
| `imageHref1` / `imageHref2` | `string`                     | Example URL       | Destinations for plain-image variants     |
| `heading1` … `subtext2`     | `string`                     | copy              | Copy used by overlay variants             |
| Color props                 | `string`                     | colors            | Canvas, surface, and overlay text colors  |
