The Feature Row component displays a feature with an icon, title, and description.

## Usage

```tsx
import { FeatureRow } from "@/registry/bases/react-email/ui/feature-row";

export default function Email() {
  return (
    <FeatureRow
      icon="🚀"
      title="Fast Setup"
      description="Get started in minutes."
    />
  );
}
```

## Props

| Prop          | Type                 | Default        | Description         |
| ------------- | -------------------- | -------------- | ------------------- |
| `theme`       | `EmailTheme`         | `defaultTheme` | Email theme styling |
| `icon`        | `string`             | -              | Icon emoji or URL   |
| `title`       | `string`             | -              | Feature title       |
| `description` | `string`             | -              | Feature description |
| `align`       | `"left" \| "center"` | `"left"`       | Text alignment      |
