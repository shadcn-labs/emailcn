## API Reference

| Prop      | Type                                                             | Default          | Description              |
| --------- | ---------------------------------------------------------------- | ---------------- | ------------------------ |
| `variant` | `"left-aligned" \| "right-aligned" \| "centered" \| "justified"` | `"left-aligned"` | Mailviews layout variant |
| `theme`   | `TailwindConfig`                                                 | `defaultTheme`   | Email theme styling      |
