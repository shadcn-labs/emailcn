## API Reference

| Prop      | Type                                                                                                                                           | Default          | Description              |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | ---------------- | ------------------------ |
| `variant` | `"left-aligned" \| "left-filled" \| "right-aligned" \| "right-filled" \| "centered" \| "centered-filled" \| "justified" \| "justified-filled"` | `"left-aligned"` | Mailviews layout variant |
| `theme`   | `TailwindConfig`                                                                                                                               | `defaultTheme`   | Email theme styling      |
