## Usage

```tsx
import { TwoColumnsReviews } from "@/components/emails/ecommerce/reviews/two-columns-reviews";
```

```tsx
<TwoColumnsReviews {...TwoColumnsReviews.PreviewProps} variant="with-divider" />
```

## API Reference

| Prop      | Type             | Default          | Description              |
| --------- | ---------------- | ---------------- | ------------------------ |
| `variant` | `ReviewsVariant` | `"with-divider"` | Mailviews layout variant |
| `theme`   | `TailwindConfig` | `defaultTheme`   | Email theme styling      |
