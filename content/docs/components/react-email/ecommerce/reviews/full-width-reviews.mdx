---
title: "Full Width Reviews"
description: "Ecommerce reviews variant for email templates"
---

Full-width reviews keep a single review across the email width while changing the arrangement of its rating, divider, logo, avatar, and author details.

## Usage

```tsx
import { FullWidthReviews } from "@/components/emails/ecommerce/reviews/full-width-reviews";
```

```tsx
<FullWidthReviews {...FullWidthReviews.PreviewProps} variant="with-divider" />
```

## API Reference

| Prop      | Type                      | Default          | Description              |
| --------- | ------------------------- | ---------------- | ------------------------ |
| `variant` | `FullWidthReviewsVariant` | `"with-divider"` | Mailviews layout variant |
| `theme`   | `TailwindConfig`          | `defaultTheme`   | Email theme styling      |
