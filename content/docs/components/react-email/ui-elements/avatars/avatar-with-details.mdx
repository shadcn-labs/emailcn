## Usage

```tsx
import { AvatarWithDetails } from "@/components/emails/ui-elements/avatars/avatar-with-details";
```

```tsx
<AvatarWithDetails align="center" />
```

## API Reference

| Prop        | Type                            | Default                   | Description                  |
| ----------- | ------------------------------- | ------------------------- | ---------------------------- |
| `align`     | `"left" \| "center" \| "right"` | `"center"`                | Positions the identity block |
| `avatarUrl` | `string`                        | Mailviews avatar URL      | Avatar image source          |
| `name`      | `string`                        | `"John Adams"`            | Display name                 |
| `email`     | `string`                        | `"johnadams@example.com"` | Email address                |
| `theme`     | Email theme type                | `defaultTheme`            | Email framework theme        |
