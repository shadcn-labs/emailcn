## API Reference

| Prop           | Type                                                                                                                                                                                              | Default                 | Description                             |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------- | --------------------------------------- |
| `variant`      | `"primary" \| "primary-leading-icon" \| "primary-trailing-icon" \| "secondary" \| "secondary-leading-icon" \| "secondary-trailing-icon" \| "text" \| "text-leading-icon" \| "text-trailing-icon"` | `"primary"`             | Selects one of the examples above       |
| `size`         | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "all"`                                                                                                                                                   | `"all"`                 | Renders one size or the five-size stack |
| `label`        | `string`                                                                                                                                                                                          | Variant-specific        | Overrides the button label              |
| `href`         | `string`                                                                                                                                                                                          | `"https://example.com"` | Button link URL                         |
| `icon`         | `ReactNode`                                                                                                                                                                                       | icon asset              | Overrides the variant's built-in icon   |
| `iconPosition` | `"leading" \| "trailing"`                                                                                                                                                                         | `"leading"`             | Position for a custom icon              |
| `align`        | `"left" \| "center" \| "right"`                                                                                                                                                                   | `"center"`              | Horizontal alignment                    |
| `theme`        | `TailwindConfig`                                                                                                                                                                                  | `defaultTheme`          | Email theme and container configuration |
