## API Reference

| Prop      | Type               | Default                   | Description                   |
| --------- | ------------------ | ------------------------- | ----------------------------- |
| `theme`   | `TailwindConfig`   | `defaultTheme`            | Renderer theme configuration. |
| `headers` | `string[]`         | Component-specific labels | Column labels.                |
| `rows`    | `Status pill rows` | One active user row       | Table row data.               |
