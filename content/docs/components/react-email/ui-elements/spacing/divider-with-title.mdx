## API Reference

| Prop      | Type                            | Default           | Description             |
| --------- | ------------------------------- | ----------------- | ----------------------- |
| `title`   | `string`                        | `"Section Title"` | Displayed title         |
| `variant` | `"center" \| "left" \| "right"` | `"center"`        | Inline content position |
| `theme`   | `TailwindConfig`                | `defaultTheme`    | Email theme styling     |
