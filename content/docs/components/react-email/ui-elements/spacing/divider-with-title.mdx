## Usage

```tsx
import { DividerWithTitle } from "@/components/emails/ui-elements/spacing/divider-with-title";
```

```tsx
<DividerWithTitle {...DividerWithTitle.PreviewProps} variant="left" />
```

## API Reference

| Prop      | Type                            | Default           | Description             |
| --------- | ------------------------------- | ----------------- | ----------------------- |
| `title`   | `string`                        | `"Section Title"` | Displayed title         |
| `variant` | `"center" \| "left" \| "right"` | `"center"`        | Inline content position |
| `theme`   | `TailwindConfig`                | `defaultTheme`    | Email theme styling     |
