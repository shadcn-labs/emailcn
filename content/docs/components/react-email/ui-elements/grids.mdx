## Usage

```tsx
import { Grid } from "@/components/email/grid";
```

```tsx
<Grid
  variant="two-columns"
  cells={["Left column", "Right column"]}
/>
<Grid
  variant="one-three-split"
  align="left"
  cells={["Sidebar", "Main content area"]}
/>
```

## API Reference

| Prop      | Type                                                                                                           | Default         | Description                       |
| --------- | -------------------------------------------------------------------------------------------------------------- | --------------- | --------------------------------- |
| `variant` | `"one-column" \| "two-columns" \| "three-columns" \| "four-columns" \| "one-three-split" \| "three-one-split"` | `"two-columns"` | Column arrangement and widths     |
| `cells`   | `string[]`                                                                                                     | `[]`            | Content for each column, in order |
| `align`   | `"left" \| "center" \| "right"`                                                                                | `"center"`      | Text alignment within columns     |
| `theme`   | `TailwindConfig`                                                                                               | `defaultTheme`  | Email theme styling               |
