## API Reference

### Logo Header

| Prop      | Type         | Default        | Description         |
| --------- | ------------ | -------------- | ------------------- |
| `theme`   | `EmailTheme` | `defaultTheme` | Email theme styling |
| `logoUrl` | `string`     | -              | Logo image URL      |
| `logoAlt` | `string`     | `"Logo"`       | Alt text for logo   |
| `href`    | `string`     | `"#"`          | Logo link URL       |
