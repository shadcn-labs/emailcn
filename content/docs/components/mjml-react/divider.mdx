The Divider component displays a horizontal line to separate sections in an email.

## Usage

```tsx
import { Divider } from "@/registry/bases/mjml-react/ui/divider";

export default function Email() {
  return <Divider />;
}
```

## Props

| Prop    | Type         | Default        | Description         |
| ------- | ------------ | -------------- | ------------------- |
| `theme` | `EmailTheme` | `defaultTheme` | Email theme styling |
