## API Reference

| Prop      | Type               | Default                   | Description                   |
| --------- | ------------------ | ------------------------- | ----------------------------- |
| `theme`   | `EmailThemeTokens` | `defaultTheme`            | Renderer theme configuration. |
| `headers` | `string[]`         | Component-specific labels | Column labels.                |
| `rows`    | `Progress rows`    | One progress row          | Table row data.               |
