## API Reference

### Pill

| Prop    | Type                                                       | Default        | Description         |
| ------- | ---------------------------------------------------------- | -------------- | ------------------- |
| `label` | `string`                                                   | -              | Pill label text     |
| `color` | `"default" \| "success" \| "warning" \| "error" \| "info"` | `"default"`    | Status color        |
| `theme` | `EmailTheme`                                               | `defaultTheme` | Email theme styling |
