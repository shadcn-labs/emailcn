## API Reference

| Prop      | Type                            | Default        | Description             |
| --------- | ------------------------------- | -------------- | ----------------------- |
| `logoSrc` | `string`                        | `undefined`    | Logo image URL          |
| `logoAlt` | `string`                        | `"Logo"`       | Logo alternative text   |
| `variant` | `"center" \| "left" \| "right"` | `"center"`     | Inline content position |
| `theme`   | `EmailThemeTokens`              | `defaultTheme` | Email theme styling     |
