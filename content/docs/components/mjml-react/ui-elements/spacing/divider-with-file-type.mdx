## Usage

```tsx
import { DividerWithFileType } from "@/components/emails/ui-elements/spacing/divider-with-file-type";
```

```tsx
<DividerWithFileType {...DividerWithFileType.PreviewProps} variant="left" />
```

## API Reference

| Prop       | Type                            | Default        | Description             |
| ---------- | ------------------------------- | -------------- | ----------------------- |
| `fileType` | `string`                        | `"PDF"`        | Displayed file type     |
| `variant`  | `"center" \| "left" \| "right"` | `"center"`     | Inline content position |
| `theme`    | `EmailThemeTokens`              | `defaultTheme` | Email theme styling     |
