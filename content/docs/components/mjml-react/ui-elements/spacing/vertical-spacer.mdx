## Usage

```tsx
import { VerticalSpacer } from "@/components/emails/ui-elements/spacing/vertical-spacer";
```

```tsx
<VerticalSpacer height={48} />
```

## API Reference

| Prop     | Type               | Default        | Description             |
| -------- | ------------------ | -------------- | ----------------------- |
| `height` | `number`           | `24`           | Spacer height in pixels |
| `theme`  | `EmailThemeTokens` | `defaultTheme` | Email theme styling     |
