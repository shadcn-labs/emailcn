## Usage

```tsx
import { DividerWithButton } from "@/components/emails/ui-elements/spacing/divider-with-button";
```

```tsx
<DividerWithButton {...DividerWithButton.PreviewProps} variant="left" />
```

## API Reference

| Prop      | Type                            | Default        | Description             |
| --------- | ------------------------------- | -------------- | ----------------------- |
| `label`   | `string`                        | `"View All"`   | Button text             |
| `href`    | `string`                        | `"#"`          | Button URL              |
| `variant` | `"center" \| "left" \| "right"` | `"center"`     | Inline content position |
| `theme`   | `EmailThemeTokens`              | `defaultTheme` | Email theme styling     |
