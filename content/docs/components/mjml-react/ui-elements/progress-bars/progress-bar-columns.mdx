## API Reference

| Prop      | Type                                          | Default        | Description                                  |
| --------- | --------------------------------------------- | -------------- | -------------------------------------------- |
| `items`   | `readonly [ProgressBarItem, ProgressBarItem]` | example items  | Two responsive progress columns              |
| `theme`   | `EmailTheme`                                  | `defaultTheme` | Email theme configuration                    |
| `variant` | `ProgressBarColumnsVariant`                   | `with-text`    | Selects content order and horizontal padding |
