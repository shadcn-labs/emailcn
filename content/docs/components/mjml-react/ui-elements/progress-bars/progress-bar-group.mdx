## API Reference

| Prop          | Type                         | Default                 | Description                                  |
| ------------- | ---------------------------- | ----------------------- | -------------------------------------------- |
| `description` | `string`                     | FlowSync description    | Supporting copy used by text variants        |
| `items`       | `readonly ProgressBarItem[]` | Mailviews example items | Vertical progress items                      |
| `theme`       | `EmailTheme`                 | `defaultTheme`          | Email theme configuration                    |
| `title`       | `string`                     | `FlowSync`              | Group heading                                |
| `variant`     | `ProgressBarPaddedVariant`   | `text-top`              | Selects content order and horizontal padding |
