## API Reference

### Button

| Prop      | Type                                 | Default        | Description          |
| --------- | ------------------------------------ | -------------- | -------------------- |
| `variant` | `"primary" \| "secondary" \| "text"` | `"primary"`    | Button style variant |
| `label`   | `string`                             | -              | Button label         |
| `href`    | `string`                             | `"#"`          | Button link URL      |
| `theme`   | `EmailTheme`                         | `defaultTheme` | Email theme styling  |
