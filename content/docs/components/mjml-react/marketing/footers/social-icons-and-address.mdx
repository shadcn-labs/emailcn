## API Reference

| Prop      | Type                                        | Default        | Description                        |
| --------- | ------------------------------------------- | -------------- | ---------------------------------- |
| `variant` | `"left-logo" \| "right-logo" \| "centered"` | `"left-logo"`  | Selects the exact Mailviews layout |
| `theme`   | `EmailTheme`                                | `defaultTheme` | Email theme styling                |
