## API Reference

| Prop          | Type                 | Default          | Description                                       |
| ------------- | -------------------- | ---------------- | ------------------------------------------------- |
| `theme`       | Renderer theme       | `defaultTheme`   | Renderer theme configuration                      |
| `variant`     | `CardCouponsVariant` | `"with-overlay"` | Selects the corresponding Mailviews coupon layout |
| Content props | `string`             | Mailviews data   | Coupon copy, code, imagery, and action content    |
| Color props   | `string`             | Mailviews colors | Canvas, card, copy, code, and action colors       |
