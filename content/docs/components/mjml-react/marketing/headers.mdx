### Usage

```tsx
import { HeaderLogoWithLinks } from "@/components/emails/marketing/headers/header-with-logo";
```

```tsx
<HeaderLogoWithLinks {...HeaderLogoWithLinks.PreviewProps} />
```

### API Reference

| Prop      | Type         | Default        | Description         |
| --------- | ------------ | -------------- | ------------------- |
| `logoUrl` | `string`     | -              | Logo image URL      |
| `logoAlt` | `string`     | `"Logo"`       | Alt text for logo   |
| `href`    | `string`     | `"#"`          | Logo link URL       |
| `theme`   | `EmailTheme` | `defaultTheme` | Email theme styling |
