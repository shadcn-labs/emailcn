## Feature with Product Image

<ComponentPreview base="mjml-react" name="feature-with-product-image-demo" />

### Installation

<CodeTabs>

<TabsList>
  <TabsTrigger value="cli">Command</TabsTrigger>
  <TabsTrigger value="manual">Manual</TabsTrigger>
</TabsList>
<TabsContent value="cli">

```bash
npx shadcn@latest add @emailcn/feature-with-product-image
```

</TabsContent>

<TabsContent value="manual">

<Steps>

<Step>Install the following dependencies:</Step>

```bash
npm install @faire/mjml-react
```

<Step>Copy the component source into your project.</Step>

<ComponentSource
  src="registry/bases/mjml-react/ui/marketing/feature/feature-with-product-image.tsx"
  base="mjml-react"
  title="components/emails/marketing/feature/feature-with-product-image.tsx"
/>

<Step>Update the import paths to match your project setup.</Step>

</Steps>

</TabsContent>

</CodeTabs>

### Usage

```tsx
import { FeatureWithProductImage } from "@/components/emails/marketing/feature/feature-with-product-image";
```

```tsx
<FeatureWithProductImage {...FeatureWithProductImage.PreviewProps} />
```

### API Reference

| Prop          | Type         | Default        | Description            |
| ------------- | ------------ | -------------- | ---------------------- |
| `imageSrc`    | `string`     | -              | Feature image URL      |
| `imageAlt`    | `string`     | -              | Feature image alt text |
| `title`       | `string`     | -              | Feature title          |
| `description` | `string`     | -              | Feature description    |
| `theme`       | `EmailTheme` | `defaultTheme` | Email theme styling    |
