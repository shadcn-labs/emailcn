```tsx

<GridStats variant="accent-column" layout="bento" />;
```

## API Reference

| Prop      | Type                                                                 | Default           | Description                      |
| --------- | -------------------------------------------------------------------- | ----------------- | -------------------------------- |
| `variant` | `"simple" \| "outlined" \| "bordered" \| "boxed" \| "accent-column"` | `"boxed"`         | visual treatment                 |
| `layout`  | `"three-columns" \| "bento" \| "bento-reversed"`                     | `"three-columns"` | Layout used by styled treatments |
| `theme`   | `EmailThemeTokens`                                                   | `defaultTheme`    | Email theme styling              |
