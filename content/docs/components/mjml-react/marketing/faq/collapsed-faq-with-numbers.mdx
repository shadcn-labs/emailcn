## API Reference

| Prop            | Type               | Default                        | Description                            |
| --------------- | ------------------ | ------------------------------ | -------------------------------------- |
| `theme`         | `EmailThemeTokens` | `defaultTheme`                 | Renderer theme configuration.          |
| `heading`       | `string`           | `"Frequently asked questions"` | Section heading.                       |
| `q1`, `q2`, ... | `string`           | Mailviews example copy         | Question labels.                       |
| `a1`, `a2`, ... | `string`           | Mailviews example copy         | Answers when the layout displays them. |
