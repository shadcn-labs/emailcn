## API Reference

| Prop                                                       | Type                        | Default           | Description                                       |
| ---------------------------------------------------------- | --------------------------- | ----------------- | ------------------------------------------------- |
| `theme`                                                    | `EmailThemeTokens`          | `defaultTheme`    | MJML React theme tokens                           |
| `variant`                                                  | `HeroBlockWithBleedVariant` | `"left-centered"` | Selects side and vertical placement               |
| `eyebrow` / `heading` / `subheading`                       | `string`                    | copy              | Main conservation copy                            |
| `description`                                              | `string`                    | copy              | Supporting paragraph                              |
| `ctaLabel` / `ctaHref`                                     | `string`                    | CTA               | Button label and destination                      |
| `backgroundImageSrc` / `imageAlt`                          | `string`                    | Landscape imagery | Full-bleed background and accessible label        |
| `logoSrc` / `logoAlt` / `logoHref`                         | `string`                    | Light logo        | Brand logo used by centered and bottom placements |
| `pageBackgroundColor` / `backgroundColor` / `overlayColor` | `string`                    | colors            | Canvas, fallback, and translucent block colors    |
| `textColor`                                                | `string`                    | `"#f9fafb"`       | Hero copy color                                   |
| `buttonBackgroundColor` / `buttonTextColor`                | `string`                    | colors            | CTA colors                                        |
