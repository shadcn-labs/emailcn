## API Reference

| Prop                                             | Type                                             | Default         | Description                                |
| ------------------------------------------------ | ------------------------------------------------ | --------------- | ------------------------------------------ |
| `theme`                                          | `EmailThemeTokens`                               | `defaultTheme`  | MJML React theme tokens                    |
| `variant`                                        | `"default" \| "slanted-left" \| "slanted-right"` | `"default"`     | Selects the exact layout and content set   |
| `eyebrow` / `heading` / `subheading`             | `string`                                         | Variant copy    | Product title copy                         |
| `description`                                    | `string`                                         | Variant copy    | Supporting product description             |
| `ctaLabel` / `ctaHref`                           | `string`                                         | CTA             | Button label and destination               |
| `backgroundImageSrc` / `imageSrc` / `imageAlt`   | `string`                                         | Variant imagery | Background and overlapping product imagery |
| `logoSrc` / `logoAlt` / `logoHref`               | `string`                                         | Light logo      | Brand logo and destination                 |
| `pageBackgroundColor` / `contentBackgroundColor` | `string`                                         | colors          | Canvas and content surface colors          |
| `textColor` / `mutedTextColor`                   | `string`                                         | Variant colors  | Primary and supporting copy colors         |
| `buttonBackgroundColor` / `buttonTextColor`      | `string`                                         | colors          | CTA colors                                 |
