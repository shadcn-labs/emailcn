## API Reference

| Prop                                             | Type                               | Default         | Description                                       |
| ------------------------------------------------ | ---------------------------------- | --------------- | ------------------------------------------------- |
| `theme`                                          | `EmailThemeTokens`                 | `defaultTheme`  | React Email Tailwind configuration                |
| `variant`                                        | `HeroWithOverlappedContentVariant` | `"basic"`       | Selects content order, imagery, copy, and surface |
| `eyebrow` / `heading` / `subheading`             | `string`                           | Variant copy    | Editorial title copy                              |
| `description`                                    | `string`                           | Variant copy    | Supporting editorial paragraph                    |
| `ctaLabel` / `ctaHref`                           | `string`                           | CTA             | Button label and destination                      |
| `backgroundImageSrc` / `imageAlt`                | `string`                           | Variant imagery | Background image and accessible label             |
| `logoSrc` / `logoAlt` / `logoHref`               | `string`                           | Light logo      | Brand logo and destination                        |
| `pageBackgroundColor` / `contentBackgroundColor` | `string`                           | Variant colors  | Canvas and solid or gradient content surface      |
| `textColor`                                      | `string`                           | Variant color   | Hero copy color                                   |
| `buttonBackgroundColor` / `buttonTextColor`      | `string`                           | colors          | CTA colors                                        |
