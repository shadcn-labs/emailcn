## API Reference

| Prop                                        | Type                                             | Default           | Description                                       |
| ------------------------------------------- | ------------------------------------------------ | ----------------- | ------------------------------------------------- |
| `theme`                                     | `EmailThemeTokens`                               | `defaultTheme`    | MJML React theme tokens                           |
| `variant`                                   | `"default" \| "slanted-left" \| "slanted-right"` | `"default"`       | Selects the flat or Mailviews slanted ending      |
| `eyebrow`                                   | `string`                                         | `"Fresh Drop"`    | Accent label above the heading                    |
| `headingAccent` / `heading`                 | `string`                                         | Mailviews copy    | Bold first word and light remaining product title |
| `subheading`                                | `string`                                         | Mailviews copy    | Supporting availability line                      |
| `description`                               | `string`                                         | Mailviews copy    | Centered product description                      |
| `ctaLabel` / `ctaHref`                      | `string`                                         | Mailviews CTA     | Button label and destination                      |
| `imageSrc` / `imageAlt`                     | `string`                                         | Mailviews imagery | Full hero background and label                    |
| `logoSrc` / `logoAlt` / `logoHref`          | `string`                                         | Mono insignia     | Centered brand mark and destination               |
| `pageBackgroundColor`                       | `string`                                         | `"#f1f5f9"`       | Email-canvas background                           |
| `slantColor`                                | `string`                                         | `"#fffffe"`       | Triangular lower-edge color                       |
| `textColor` / `accentColor`                 | `string`                                         | Mailviews colors  | Main and accent text colors                       |
| `buttonBackgroundColor` / `buttonTextColor` | `string`                                         | Mailviews colors  | CTA colors                                        |
