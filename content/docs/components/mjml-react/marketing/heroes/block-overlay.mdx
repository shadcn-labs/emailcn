## API Reference

| Prop                                                       | Type                      | Default           | Description                                      |
| ---------------------------------------------------------- | ------------------------- | ----------------- | ------------------------------------------------ |
| `theme`                                                    | `EmailThemeTokens`        | `defaultTheme`    | React Email Tailwind configuration               |
| `variant`                                                  | `HeroBlockOverlayVariant` | `"left-centered"` | Selects overlay side and vertical placement      |
| `eyebrow` / `heading` / `subheading`                       | `string`                  | copy              | Main SumUp copy                                  |
| `description`                                              | `string`                  | copy              | Supporting paragraph                             |
| `ctaLabel` / `ctaHref`                                     | `string`                  | CTA               | Button label and destination                     |
| `backgroundImageSrc` / `imageAlt`                          | `string`                  | SumUp imagery     | Full-bleed background and accessible label       |
| `logoSrc` / `logoAlt` / `logoHref`                         | `string`                  | Light logo        | Logo used by centered and bottom placements      |
| `pageBackgroundColor` / `backgroundColor` / `overlayColor` | `string`                  | colors            | Canvas, fallback, and translucent overlay colors |
| `textColor`                                                | `string`                  | `"#f9fafb"`       | Hero copy color                                  |
| `buttonBackgroundColor` / `buttonTextColor`                | `string`                  | colors            | CTA colors                                       |
