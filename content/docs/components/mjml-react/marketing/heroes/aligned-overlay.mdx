## API Reference

| Prop                                              | Type                                                                                       | Default          | Description                                            |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------ | ---------------- | ------------------------------------------------------ |
| `theme`                                           | `EmailThemeTokens`                                                                         | `defaultTheme`   | MJML React theme tokens                                |
| `variant`                                         | `"content-left" \| "content-left-reversed" \| "content-right" \| "content-right-reversed"` | `"content-left"` | Selects image edge, copy alignment, and vertical order |
| `eyebrow`                                         | `string`                                                                                   | copy             | Small line above the heading                           |
| `headingStart` / `headingEmphasis` / `headingEnd` | `string`                                                                                   | copy             | Three heading segments with a lighter middle segment   |
| `description`                                     | `string`                                                                                   | copy             | Supporting travel copy                                 |
| `ctaLabel` / `ctaHref`                            | `string`                                                                                   | CTA              | Button label and destination                           |
| `imageSrc` / `imageAlt`                           | `string`                                                                                   | Variant imagery  | Edge-aligned background image and label                |
| `logoSrc` / `logoAlt` / `logoHref`                | `string`                                                                                   | Light logo       | Brand logo and destination                             |
| `pageBackgroundColor` / `backgroundColor`         | `string`                                                                                   | colors           | Email canvas and dark hero surface                     |
| `textColor`                                       | `string`                                                                                   | `"#f9fafb"`      | Hero copy color                                        |
| `buttonBackgroundColor` / `buttonTextColor`       | `string`                                                                                   | colors           | CTA colors                                             |
