## API Reference

| Prop                    | Type                                                                               | Default           | Description                                                    |
| ----------------------- | ---------------------------------------------------------------------------------- | ----------------- | -------------------------------------------------------------- |
| `theme`                 | `EmailThemeTokens`                                                                 | `defaultTheme`    | MJML React theme tokens                                        |
| `variant`               | `"images-bottom" \| "images-top" \| "offset-images-bottom" \| "offset-images-top"` | `"images-bottom"` | Selects the image placement and mosaic style                   |
| `eyebrow`               | `string`                                                                           | `"NEW ARRIVALS"`  | Small label above the heading                                  |
| `heading`               | `string`                                                                           | `"Step into…"`    | Main hero heading                                              |
| `subheading`            | `string`                                                                           | copy              | Supporting line below the heading                              |
| `description`           | `string`                                                                           | copy              | Centered supporting paragraph                                  |
| `images`                | `HeroWithImageGridImage[]`                                                         | imagery           | Overrides the first 7 regular or first 10 offset mosaic images |
| `logoSrc` / `logoAlt`   | `string`                                                                           | logo              | Brand logo shown above the hero                                |
| `logoHref`              | `string`                                                                           | Example URL       | Logo destination                                               |
| `ctaLabel` / `ctaHref`  | `string`                                                                           | CTA               | Button label and destination                                   |
| `pageBackgroundColor`   | `string`                                                                           | `"#f1f5f9"`       | Email-canvas background                                        |
| `backgroundColor`       | `string`                                                                           | `"#fffffe"`       | Hero background                                                |
| `buttonBackgroundColor` | `string`                                                                           | `"#4f46e5"`       | CTA background                                                 |
| `buttonTextColor`       | `string`                                                                           | `"#fffffe"`       | CTA text color                                                 |
