## API Reference

| Prop                                                                      | Type                          | Default               | Description                                     |
| ------------------------------------------------------------------------- | ----------------------------- | --------------------- | ----------------------------------------------- |
| `theme`                                                                   | `EmailThemeTokens`            | `defaultTheme`        | React Email Tailwind configuration              |
| `variant`                                                                 | `HeroWithSlantedSplitVariant` | `"left-slanted-down"` | Selects image side and slant direction          |
| `eyebrow` / `heading` / `subheading`                                      | `string`                      | copy                  | Magazine title copy                             |
| `description`                                                             | `string`                      | copy                  | Supporting editorial paragraph                  |
| `ctaLabel` / `ctaHref`                                                    | `string`                      | CTA                   | Button label and destination                    |
| `imageSrc` / `imageAlt`                                                   | `string`                      | imagery               | Full-height image and accessible label          |
| `logoSrc` / `logoAlt` / `logoHref`                                        | `string`                      | Light logo            | Brand logo and destination                      |
| `pageBackgroundColor` / `imageBackgroundColor` / `contentBackgroundColor` | `string`                      | colors                | Canvas, image fallback, and copy surface colors |
| `textColor`                                                               | `string`                      | `"#030712"`           | Hero copy color                                 |
| `buttonBackgroundColor` / `buttonTextColor`                               | `string`                      | colors                | CTA colors                                      |
