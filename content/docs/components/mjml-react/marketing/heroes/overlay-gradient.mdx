## API Reference

| Prop                                                       | Type                             | Default             | Description                                   |
| ---------------------------------------------------------- | -------------------------------- | ------------------- | --------------------------------------------- |
| `theme`                                                    | `EmailThemeTokens`               | `defaultTheme`      | React Email Tailwind configuration            |
| `variant`                                                  | `HeroWithOverlayGradientVariant` | `"split-with-logo"` | Selects copy grouping and logo visibility     |
| `eyebrow` / `heading` / `subheading`                       | `string`                         | copy                | Product title copy                            |
| `price` / `description`                                    | `string`                         | copy                | Price and supporting product description      |
| `ctaLabel` / `ctaHref`                                     | `string`                         | CTA                 | Button label and destination                  |
| `backgroundImageSrc` / `imageAlt`                          | `string`                         | Yosemite imagery    | Full-bleed background and accessible label    |
| `logoSrc` / `logoAlt` / `logoHref`                         | `string`                         | Light logo          | Optional brand logo and destination           |
| `pageBackgroundColor` / `backgroundColor` / `overlayColor` | `string`                         | colors              | Canvas, fallback, and gradient surface colors |
| `textColor`                                                | `string`                         | `"#f9fafb"`         | Hero copy color                               |
| `buttonBackgroundColor` / `buttonTextColor`                | `string`                         | colors              | CTA colors                                    |
