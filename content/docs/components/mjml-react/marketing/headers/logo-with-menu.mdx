## Usage

```tsx
import { HeaderWithLogoAndMenu } from "@/components/emails/marketing/headers/header-with-logo-and-menu";
```

```tsx
<HeaderWithLogoAndMenu
  {...HeaderWithLogoAndMenu.PreviewProps}
  variant="menu-around"
/>
```

## API Reference

| Prop      | Type                           | Default        | Description                       |
| --------- | ------------------------------ | -------------- | --------------------------------- |
| `variant` | `HeaderWithLogoAndMenuVariant` | `"menu-right"` | Selects the Mailviews menu layout |
| `theme`   | Renderer theme                 | `defaultTheme` | Renderer theme configuration      |
