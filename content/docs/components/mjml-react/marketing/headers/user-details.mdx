## API Reference

| Prop        | Type                             | Default        | Description                      |
| ----------- | -------------------------------- | -------------- | -------------------------------- |
| `avatar`    | `HeaderWithUserDetailsAvatar`    | `"initials"`   | Selects initials or photo avatar |
| `alignment` | `HeaderWithUserDetailsAlignment` | `"left"`       | Places the logo left or right    |
| `theme`     | Renderer theme                   | `defaultTheme` | Renderer theme configuration     |
