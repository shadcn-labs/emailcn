## Usage

```tsx

```

```tsx
<HeaderWithLogo
  {...HeaderWithLogo.PreviewProps}
  alignment="right"
  variant="with-text"
/>
```

## API Reference

| Prop        | Type                      | Default        | Description                       |
| ----------- | ------------------------- | -------------- | --------------------------------- |
| `variant`   | `HeaderWithLogoVariant`   | `"minimal"`    | Selects minimal or text content   |
| `alignment` | `HeaderWithLogoAlignment` | `"center"`     | Places the logo left/center/right |
| `theme`     | Renderer theme            | `defaultTheme` | Renderer theme configuration      |
