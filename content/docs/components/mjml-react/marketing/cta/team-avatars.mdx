## API Reference

| Prop                                     | Type                                            | Default           | Description                                             |
| ---------------------------------------- | ----------------------------------------------- | ----------------- | ------------------------------------------------------- |
| `theme`                                  | `EmailThemeTokens`                              | `defaultTheme`    | React Email Tailwind configuration                      |
| `variant`                                | `"default" \| "avatars-top" \| "single-avatar"` | `"default"`       | Selects the exact layout and copy                       |
| `heading` / `subtext`                    | `string`                                        | Variant copy      | CTA heading and supporting text                         |
| `ctaLabel` / `ctaHref`                   | `string`                                        | Variant CTA       | Primary action label and destination                    |
| `secondaryCtaLabel` / `secondaryCtaHref` | `string`                                        | Single-avatar CTA | Optional secondary action                               |
| Avatar props                             | `string`                                        | avatars           | Four avatar URLs and alternative-text values            |
| Color props                              | `string`                                        | colors            | Canvas, surface, avatar border, text, and button colors |
