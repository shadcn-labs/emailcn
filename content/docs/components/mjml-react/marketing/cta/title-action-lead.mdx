## API Reference

| Prop                                                                                         | Type                               | Default            | Description                          |
| -------------------------------------------------------------------------------------------- | ---------------------------------- | ------------------ | ------------------------------------ |
| `theme`                                                                                      | `EmailThemeTokens`                 | `defaultTheme`     | React Email Tailwind configuration   |
| `variant`                                                                                    | `CTAWithTitleAndActionLeadVariant` | `"title-and-lead"` | Selects the exact CTA layout         |
| `heading` / `subtext`                                                                        | `string`                           | copy               | Lead heading and supporting text     |
| `ctaLabel` / `ctaHref`                                                                       | `string`                           | Variant CTA        | Primary button label and destination |
| `secondaryCtaLabel` / `secondaryCtaHref`                                                     | `string`                           | secondary          | Optional secondary action            |
| `signoff`                                                                                    | `string`                           | signoff            | Copy beneath the minimal CTA         |
| `pageBackgroundColor` / `backgroundColor`                                                    | `string`                           | colors             | Canvas and CTA surface colors        |
| `headingColor` / `textColor`                                                                 | `string`                           | colors             | Heading and supporting text colors   |
| `primaryButtonBackgroundColor` / `primaryButtonTextColor`                                    | `string`                           | colors             | Primary button colors                |
| `secondaryButtonBackgroundColor` / `secondaryButtonTextColor` / `secondaryButtonBorderColor` | `string`                           | colors             | Secondary button colors and border   |
