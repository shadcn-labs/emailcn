## API Reference

| Prop                                     | Type                             | Default         | Description                                       |
| ---------------------------------------- | -------------------------------- | --------------- | ------------------------------------------------- |
| `theme`                                  | `EmailThemeTokens`               | `defaultTheme`  | MJML React theme tokens                           |
| `variant`                                | `"flush" \| "boxed" \| "padded"` | `"flush"`       | Selects the exact layout and copy                 |
| `heading` / `subtext` / `emphasis`       | `string`                         | Variant content | CTA copy; `emphasis` renders bold inline text     |
| `ctaLabel` / `ctaHref`                   | `string`                         | Variant CTA     | Primary action label and destination              |
| `secondaryCtaLabel` / `secondaryCtaHref` | `string`                         | Variant CTA     | Secondary action label and destination            |
| `backgroundSrc`                          | `string`                         | Variant image   | Background image URL                              |
| Color props                              | `string`                         | colors          | Canvas, surface, overlay, text, and button colors |
