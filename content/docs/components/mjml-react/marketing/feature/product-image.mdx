## API Reference

| Prop          | Type                             | Default        | Description                                       |
| ------------- | -------------------------------- | -------------- | ------------------------------------------------- |
| `theme`       | Renderer theme                   | `defaultTheme` | Renderer theme configuration                      |
| `variant`     | `FeatureWithProductImageVariant` | `"image-left"` | Places the product image before or after the copy |
| Content props | `string`                         | data           | Heading, body, button, image, and arrow content   |
| Color props   | `string`                         | colors         | Canvas, surface, copy, and link colors            |
