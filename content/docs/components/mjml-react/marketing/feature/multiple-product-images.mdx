## API Reference

| Prop          | Type                                      | Default          | Description                                               |
| ------------- | ----------------------------------------- | ---------------- | --------------------------------------------------------- |
| `theme`       | Renderer theme                            | `defaultTheme`   | Renderer theme configuration                              |
| `variant`     | `FeatureWithMultipleProductImagesVariant` | `"logo-left"`    | Selects the Mailviews logo or image arrangement           |
| Content props | `string`                                  | Mailviews data   | Heading, body, action, logo, and product image content    |
| Color props   | `string`                                  | Mailviews colors | Canvas, surface, logo panel, copy, and action link colors |
