## API Reference

| Prop                | Type                                  | Default          | Description                              |
| ------------------- | ------------------------------------- | ---------------- | ---------------------------------------- |
| `theme`             | Renderer theme                        | `defaultTheme`   | Renderer theme configuration             |
| `variant`           | `ThreeColumnsMasonryImageGridVariant` | `"stacked-left"` | Selects stack side and overlay mode      |
| Stacked image props | `string`                              | data             | Two stacked images, links, and copy      |
| Wide image props    | `string`                              | data             | Wide image, link, alternative text, copy |
| Color props         | `string`                              | colors           | Canvas, surface, and overlay text colors |
