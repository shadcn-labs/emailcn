## API Reference

| Prop                         | Type                               | Default        | Description                                     |
| ---------------------------- | ---------------------------------- | -------------- | ----------------------------------------------- |
| `theme`                      | `EmailThemeTokens`                 | `defaultTheme` | React Email Tailwind configuration              |
| `variant`                    | `FullWidthImageWithOverlayVariant` | `"default"`    | Selects the exact edge treatment                |
| `imageSrc`                   | `string`                           | image          | Background image URL                            |
| `heading` / `subtext`        | `string`                           | copy           | Bottom overlay heading and supporting text      |
| `headingColor` / `textColor` | `string`                           | `"#fffffe"`    | Overlay copy colors                             |
| `pageBackgroundColor`        | `string`                           | `"#f1f5f9"`    | Canvas and cutout color                         |
| `spacerBackgroundColor`      | `string`                           | `"#fffffe"`    | Color used for padding and side spacer surfaces |
