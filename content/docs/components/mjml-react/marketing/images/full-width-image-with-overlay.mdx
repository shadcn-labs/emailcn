## API Reference

| Prop                         | Type                               | Default         | Description                                     |
| ---------------------------- | ---------------------------------- | --------------- | ----------------------------------------------- |
| `theme`                      | `EmailThemeTokens`                 | `defaultTheme`  | MJML React theme tokens                         |
| `variant`                    | `FullWidthImageWithOverlayVariant` | `"default"`     | Selects the exact Mailviews edge treatment      |
| `imageSrc`                   | `string`                           | Mailviews image | Background image URL                            |
| `heading` / `subtext`        | `string`                           | Mailviews copy  | Bottom overlay heading and supporting text      |
| `headingColor` / `textColor` | `string`                           | `"#fffffe"`     | Overlay copy colors                             |
| `pageBackgroundColor`        | `string`                           | `"#f1f5f9"`     | Canvas and cutout color                         |
| `spacerBackgroundColor`      | `string`                           | `"#fffffe"`     | Color used for padding and side spacer surfaces |
