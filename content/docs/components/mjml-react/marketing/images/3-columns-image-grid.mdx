## API Reference

| Prop        | Type                           | Default           | Description                                   |
| ----------- | ------------------------------ | ----------------- | --------------------------------------------- |
| `theme`     | Renderer theme                 | `defaultTheme`    | Renderer theme configuration                  |
| `variant`   | `ThreeColumnsImageGridVariant` | `"square-images"` | Selects shape and overlay mode                |
| Image props | `string`                       | data              | Three image links, alternative text, and copy |
| Color props | `string`                       | colors            | Canvas, surface, and overlay text colors      |
