## API Reference

| Prop      | Type                                                                                      | Default           | Description              |
| --------- | ----------------------------------------------------------------------------------------- | ----------------- | ------------------------ |
| `variant` | `"rating-bottom" \| "rating-top" \| "header-top" \| "rating-aside" \| "rating-aside-top"` | `"rating-bottom"` | Mailviews layout variant |
| `theme`   | `EmailThemeTokens`                                                                        | `defaultTheme`    | Email theme styling      |
