---
title: "Masonry Grid Reviews"
description: "Ecommerce reviews variant for email templates"
---

Masonry reviews pair one featured review with two supporting review columns. All Mailviews treatments remain variants of that same responsive structure.

## Usage

```tsx
import { MasonryGridReviews } from "@/components/emails/ecommerce/reviews/masonry-grid-reviews";
```

```tsx
<MasonryGridReviews
  {...MasonryGridReviews.PreviewProps}
  variant="with-divider"
/>
```

## API Reference

| Prop      | Type               | Default          | Description              |
| --------- | ------------------ | ---------------- | ------------------------ |
| `variant` | `ReviewsVariant`   | `"with-divider"` | Mailviews layout variant |
| `theme`   | `EmailThemeTokens` | `defaultTheme`   | Email theme styling      |
