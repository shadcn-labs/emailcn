## Usage

```tsx
import { ShoppingCartRowItems } from "@/components/emails/ecommerce/shopping-cart/shopping-cart-with-row-items";
```

```tsx
<ShoppingCartRowItems {...ShoppingCartRowItems.PreviewProps} variant="basic" />
```

## API Reference

| Prop      | Type                  | Default        | Description              |
| --------- | --------------------- | -------------- | ------------------------ |
| `variant` | `ShoppingCartVariant` | `"basic"`      | Mailviews layout variant |
| `items`   | `ShoppingCartItem[]`  | Mailviews data | Cart item overrides      |
| `theme`   | `EmailThemeTokens`    | `defaultTheme` | Email theme styling      |
