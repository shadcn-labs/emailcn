## API Reference

| Prop      | Type                                                                                                                                                                         | Default        | Description          |
| --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------- | -------------------- |
| `variant` | `"muted" \| "muted-reverse" \| "boxed" \| "boxed-reverse" \| "accent" \| "accent-reverse" \| "image-top" \| "image-top-reverse" \| "image-bottom" \| "image-bottom-reverse"` | `"muted"`      | split-card treatment |
| `theme`   | `EmailThemeTokens`                                                                                                                                                           | `defaultTheme` | Email theme styling  |
