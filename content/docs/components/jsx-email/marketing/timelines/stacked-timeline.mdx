## API Reference

| Prop      | Type                                                                                                                                         | Default        | Description                |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------------- | -------------- | -------------------------- |
| `layout`  | `"line" \| "boxed"`                                                                                                                          | `"line"`       | timeline family            |
| `variant` | `"muted-left" \| "muted-right" \| "basic-left" \| "basic-right" \| "completed-left" \| "completed-right" \| "accent-left" \| "accent-right"` | `"muted-left"` | Layout state and alignment |
| `theme`   | `EmailThemeTokens`                                                                                                                           | `defaultTheme` | Email theme styling        |
