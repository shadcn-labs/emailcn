## Usage

```tsx

```

```tsx
<StackedTimeline {...StackedTimeline.PreviewProps} layout="boxed" />
```

## API Reference

| Prop      | Type                                             | Default        | Description          |
| --------- | ------------------------------------------------ | -------------- | -------------------- |
| `layout`  | `"line" \| "boxed"`                              | `"line"`       | Timeline item style  |
| `variant` | `"default" \| "slanted-left" \| "slanted-right"` | `"default"`    | Section tilt         |
| `theme`   | `EmailThemeTokens`                                 | `defaultTheme` | Email theme styling  |
