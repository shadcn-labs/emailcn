## Usage

```tsx
import { SocialsWithTileLabels } from "@/components/emails/marketing/social/socials-with-tile-labels";

<SocialsWithTileLabels variant="inline" />;
```

## API Reference

| Prop          | Type                    | Default             | Description                 |
| ------------- | ----------------------- | ------------------- | --------------------------- |
| `variant`     | `"stacked" \| "inline"` | `"stacked"`         | Label arrangement           |
| `items`       | `TiledSocialItem[]`     | Mailviews icons     | Social links, labels, icons |
| `title`       | `string`                | `"Connect with us"` | Section title               |
| `description` | `string`                | Mailviews copy      | Supporting text             |
| `theme`       | `EmailThemeTokens`      | `defaultTheme`      | Email theme configuration   |
