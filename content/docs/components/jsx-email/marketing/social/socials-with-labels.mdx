## Usage

```tsx
import { SocialsWithLabels } from "@/components/emails/marketing/social/socials-with-labels";

<SocialsWithLabels variant="inline" />;
```

## API Reference

| Prop          | Type                    | Default             | Description                 |
| ------------- | ----------------------- | ------------------- | --------------------------- |
| `variant`     | `"stacked" \| "inline"` | `"stacked"`         | Label arrangement           |
| `items`       | `LabeledSocialItem[]`   | Mailviews icons     | Social links, labels, icons |
| `title`       | `string`                | `"Connect with us"` | Section title               |
| `description` | `string`                | Mailviews copy      | Supporting text             |
| `theme`       | `EmailThemeTokens`      | `defaultTheme`      | Email theme configuration   |
