
## API Reference

| Prop      | Type                                                          | Default        | Description              |
| --------- | ------------------------------------------------------------- | -------------- | ------------------------ |
| `variant` | `"default" \| "three-columns" \| "bento" \| "bento-reversed"` | `"default"`    | Mailviews layout variant |
| `theme`   | `EmailThemeTokens`                                            | `defaultTheme` | Email theme styling      |
