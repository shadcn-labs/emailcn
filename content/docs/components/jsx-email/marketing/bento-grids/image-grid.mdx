## API Reference

| Prop        | Type                                                                     | Default          | Description                             |
| ----------- | ------------------------------------------------------------------------ | ---------------- | --------------------------------------- |
| `variant`   | `"alternating" \| "three-column"`                                        | `"alternating"`  | Selects the image-grid composition      |
| `style`     | `"captions" \| "details" \| "flush" \| "padded"`                         | `"captions"`     | Selects the treatment valid for variant |
| `placement` | `BentoCaptionsVariant \| BentoDetailsVariant \| BentoThreeColumnVariant` | `"captions-top"` | Controls caption placement and ordering |
| `data`      | `AlternatingImageData \| ThreeColumnData`                                | Preset content   | Replaces the typed content for the grid |
| `theme`     | `EmailThemeTokens`                                                       | `defaultTheme`   | Controls the email theme                |
