## API Reference

| Prop        | Type                                             | Default             | Description                                |
| ----------- | ------------------------------------------------ | ------------------- | ------------------------------------------ |
| `variant`   | `"two-thirds" \| "even-split" \| "three-column"` | `"two-thirds"`      | Selects the stats composition              |
| `style`     | `"compact" \| "chart" \| "text"`                 | `"compact"`         | Selects a treatment valid for the variant  |
| `placement` | `BentoImagePlacementVariant`                     | `"image-top-right"` | Controls image position and row ordering   |
| `data`      | Variant-specific typed stats data                | Preset content      | Replaces image, metric, feature, and stats |
| `theme`     | `EmailThemeTokens`                               | `defaultTheme`      | Controls the email theme                   |
