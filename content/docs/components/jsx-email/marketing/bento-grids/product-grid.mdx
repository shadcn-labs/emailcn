## API Reference

| Prop        | Type                                                | Default                  | Description                     |
| ----------- | --------------------------------------------------- | ------------------------ | ------------------------------- |
| `variant`   | `"two-row-three-column" \| "even-split-two-thirds"` | `"two-row-three-column"` | Selects the product composition |
| `style`     | `"left" \| "right" \| "sides" \| "full"`            | `"full"`                 | Controls image padding          |
| `placement` | `"normal" \| "reverse"`                             | `"normal"`               | Controls row ordering           |
| `data`      | Variant-specific product tuples                     | Preset content           | Replaces product content        |
| `theme`     | `EmailThemeTokens`                                  | `defaultTheme`           | Controls the email theme        |
