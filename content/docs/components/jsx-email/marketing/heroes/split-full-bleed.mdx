## API Reference

| Prop                    | Type                            | Default         | Description                                      |
| ----------------------- | ------------------------------- | --------------- | ------------------------------------------------ |
| `theme`                 | `EmailThemeTokens`              | `defaultTheme`  | React Email Tailwind configuration               |
| `variant`               | `"image-left" \| "image-right"` | `"image-left"`  | Places the full-bleed image on the selected side |
| `eyebrow`               | `string`                        | `"Coffee"`      | Small label above the heading                    |
| `heading`               | `string`                        | `"Sarina"`      | Main product heading                             |
| `subheading`            | `string`                        | `"Americano…"`  | Supporting line below the heading                |
| `description`           | `string`                        | copy            | Product description                              |
| `price`                 | `string`                        | `"From $14.99"` | Bold price line; pass an empty string to hide it |
| `ctaLabel` / `ctaHref`  | `string`                        | CTA             | Button label and destination                     |
| `imageSrc` / `imageAlt` | `string`                        | imagery         | Full-bleed background image and accessible label |
| `logoSrc` / `logoAlt`   | `string`                        | logo            | Logo displayed over the image                    |
| `textBackgroundColor`   | `string`                        | `"#fffffe"`     | Text-panel background                            |
| `pageBackgroundColor`   | `string`                        | `"#f1f5f9"`     | Email-canvas background                          |
| `buttonBackgroundColor` | `string`                        | `"#4f46e5"`     | CTA background                                   |
| `buttonTextColor`       | `string`                        | `"#fffffe"`     | CTA text color                                   |
