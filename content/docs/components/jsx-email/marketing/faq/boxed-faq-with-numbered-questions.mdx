## API Reference

| Prop            | Type               | Default                        | Description                            |

| `heading`       | `string`           | `"Frequently asked questions"` | Section heading.                       |
| `q1`, `q2`, ... | `string`           | example copy                   | Question labels.                       |
| `a1`, `a2`, ... | `string`           | example copy                   | Answers when the layout displays them. |
