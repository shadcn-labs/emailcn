## API Reference

| Prop          | Type                                           | Default           | Description                                         |
| ------------- | ---------------------------------------------- | ----------------- | --------------------------------------------------- |
| `theme`       | Renderer theme                                 | `defaultTheme`    | Renderer theme configuration                        |
| `variant`     | `FeatureWithDoubleTallBackgroundImagesVariant` | `"logo-top-left"` | Selects logo placement and left/right column order  |
| Content props | `string`                                       | data              | Heading, body, action, logo, and background imagery |
| Color props   | `string`                                       | colors            | Canvas, surface, image, logo, copy, and link colors |
