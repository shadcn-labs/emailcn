## API Reference

| Prop          | Type                                      | Default        | Description                                               |
| ------------- | ----------------------------------------- | -------------- | --------------------------------------------------------- |
| `theme`       | Renderer theme                            | `defaultTheme` | Renderer theme configuration                              |
| `variant`     | `FeatureWithMultipleProductImagesVariant` | `"logo-left"`  | Selects the logo or image arrangement                     |
| Content props | `string`                                  | data           | Heading, body, action, logo, and product image content    |
| Color props   | `string`                                  | colors         | Canvas, surface, logo panel, copy, and action link colors |
