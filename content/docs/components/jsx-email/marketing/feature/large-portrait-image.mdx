## API Reference

| Prop          | Type                                   | Default           | Description                                          |
| ------------- | -------------------------------------- | ----------------- | ---------------------------------------------------- |
| `theme`       | Renderer theme                         | `defaultTheme`    | Renderer theme configuration                         |
| `variant`     | `FeatureWithLargePortraitImageVariant` | `"logo-top-left"` | Selects the media, copy, and column layout           |
| Content props | `string`                               | data              | Heading, body, action, logo, and portrait image data |
| Color props   | `string`                               | colors            | Canvas, surface, logo panel, copy, and action colors |
