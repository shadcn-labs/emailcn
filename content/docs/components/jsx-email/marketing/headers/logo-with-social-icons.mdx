| Prop        | Type                                    | Default        | Description                             |

| `alignment` | `HeaderWithLogoAndSocialIconsAlignment` | `"left"`       | Places the logo and social icon content |
| `theme`     | Renderer theme                          | `defaultTheme` | Renderer theme configuration            |
