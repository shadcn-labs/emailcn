## Usage

```tsx
import { HeaderWithLogoAndFinanceStats } from "@/components/emails/marketing/headers/header-with-logo-and-finance-stats";
```

```tsx
<HeaderWithLogoAndFinanceStats
  {...HeaderWithLogoAndFinanceStats.PreviewProps}
  alignment="center"
/>
```

## API Reference

| Prop        | Type                                     | Default        | Description                              |
| ----------- | ---------------------------------------- | -------------- | ---------------------------------------- |
| `alignment` | `HeaderWithLogoAndFinanceStatsAlignment` | `"left"`       | Places the logo and finance stat content |
| `theme`     | Renderer theme                           | `defaultTheme` | Renderer theme configuration             |
