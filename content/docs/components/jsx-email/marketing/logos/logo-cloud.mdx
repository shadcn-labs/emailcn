## API Reference

| Prop      | Type               | Default        | Description                     |
| --------- | ------------------ | -------------- | ------------------------------- |
| `tone`    | `LogoCloudTone`    | `"boxed"`      | Selects boxed or outlined tiles |
| `variant` | `LogoCloudVariant` | `"full"`       | Selects the content arrangement |
| `theme`   | Renderer theme     | `defaultTheme` | Renderer theme configuration    |
