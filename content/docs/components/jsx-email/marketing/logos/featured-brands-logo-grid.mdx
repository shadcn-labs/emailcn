## API Reference

| Prop        | Type                              | Default        | Description                                 |
| ----------- | --------------------------------- | -------------- | ------------------------------------------- |
| `tone`      | `FeaturedBrandsLogoGridTone`      | `"outlined"`   | Selects outlined or boxed tiles             |
| `alignment` | `FeaturedBrandsLogoGridAlignment` | `"left"`       | Places the featured brand left/center/right |
| `theme`     | Renderer theme                    | `defaultTheme` | Renderer theme configuration                |
