## API Reference

| Prop                      | Type                          | Default               | Description                                 |
| ------------------------- | ----------------------------- | --------------------- | ------------------------------------------- |
| `theme`                   | `EmailThemeTokens`            | `defaultTheme`        | React Email Tailwind configuration          |
| `variant`                 | `CTAWithShiftedImagesVariant` | `"flush-side-images"` | Selects the exact layout and assets         |
| `heading` / `subtext`     | `string`                      | Variant copy          | CTA heading and supporting text             |
| `ctaLabel` / `ctaHref`    | `string`                      | CTA                   | Primary action label and destination        |
| `imageSrc1` … `imageSrc4` | `string`                      | Variant images        | Image URLs used by the selected composition |
| `imageAlt1` … `imageAlt4` | `string`                      | `""`                  | Alternative text for each image             |
| Color props               | `string`                      | colors                | Canvas, surface, text, and button colors    |
