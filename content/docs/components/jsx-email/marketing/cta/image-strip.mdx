## API Reference

| Prop                      | Type                       | Default          | Description                                          |
| ------------------------- | -------------------------- | ---------------- | ---------------------------------------------------- |
| `theme`                   | `EmailThemeTokens`         | `defaultTheme`   | JSX Email theme tokens                               |
| `variant`                 | `CTAWithImageStripVariant` | `"boxed-right"`  | Selects the exact Mailviews side and strip treatment |
| `heading` / `subtext`     | `string`                   | Mailviews copy   | CTA heading and supporting text                      |
| `ctaLabel` / `ctaHref`    | `string`                   | Mailviews CTA    | Primary action label and destination                 |
| `imageSrc1` … `imageSrc5` | `string`                   | Variant images   | Image URLs used by the strip                         |
| `imageAlt1` … `imageAlt5` | `string`                   | `""`             | Alternative text for each strip image                |
| Color props               | `string`                   | Mailviews colors | Canvas, surface, strip, text, and button colors      |
