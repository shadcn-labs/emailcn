## API Reference

| Prop                        | Type                                           | Default          | Description                               |
| --------------------------- | ---------------------------------------------- | ---------------- | ----------------------------------------- |
| `theme`                     | `EmailThemeTokens`                             | `defaultTheme`   | React Email Tailwind configuration        |
| `variant`                   | `TwoColumnsMasonryImageGridWith3ImagesVariant` | `"stacked-left"` | Selects side and overlay treatment        |
| `imageSrc1` … `imageSrc3`   | `string`                                       | images           | Landscape and portrait image URLs         |
| `imageAlt1` … `imageAlt3`   | `string`                                       | `""`             | Alternative text for plain-image variants |
| `imageHref1` … `imageHref3` | `string`                                       | Example URL      | Destinations for plain-image variants     |
| `heading1` … `subtext3`     | `string`                                       | copy             | Copy used by overlay variants             |
| Color props                 | `string`                                       | colors           | Canvas, surface, and overlay text colors  |
