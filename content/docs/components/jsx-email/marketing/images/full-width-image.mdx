## API Reference

| Prop                    | Type                    | Default        | Description                                     |
| ----------------------- | ----------------------- | -------------- | ----------------------------------------------- |
| `theme`                 | `EmailThemeTokens`      | `defaultTheme` | React Email Tailwind configuration              |
| `variant`               | `FullWidthImageVariant` | `"default"`    | Selects the exact edge treatment                |
| `imageSrc` / `imageAlt` | `string`                | image          | Image URL and alternative text                  |
| `pageBackgroundColor`   | `string`                | `"#f1f5f9"`    | Canvas and cutout color                         |
| `spacerBackgroundColor` | `string`                | `"#fffffe"`    | Color used for padding and side spacer surfaces |
