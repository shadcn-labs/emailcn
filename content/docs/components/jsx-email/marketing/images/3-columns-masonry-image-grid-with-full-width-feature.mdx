## API Reference

| Prop                | Type                                                      | Default          | Description                                     |
| ------------------- | --------------------------------------------------------- | ---------------- | ----------------------------------------------- |
| `theme`             | Renderer theme                                            | `defaultTheme`   | Renderer theme configuration                    |
| `variant`           | `ThreeColumnsMasonryImageGridWithFullWidthFeatureVariant` | `"stacked-left"` | Selects side, feature order, and overlay mode   |
| Feature image props | `string`                                                  | data             | Full-width feature image, link, and copy        |
| Masonry image props | `string`                                                  | data             | Two stacked images and one wide image with copy |
| Color props         | `string`                                                  | colors           | Canvas, surface, and overlay text colors        |
