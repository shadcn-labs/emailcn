## API Reference

| Prop                | Type                                             | Default            | Description                                     |
| ------------------- | ------------------------------------------------ | ------------------ | ----------------------------------------------- |
| `theme`             | `TailwindConfig` / `EmailThemeTokens`            | `defaultTheme`     | Renderer theme configuration                    |
| `variant`           | `TwoColumnsImageGridWithFullWidthFeatureVariant` | `"full-width-top"` | Selects feature order and overlay mode          |
| Feature image props | `string`                                         | data               | Feature image, link, alternative text, and copy |
| Column image props  | `string`                                         | data               | Two image links, alternative text, and copy     |
| Color props         | `string`                                         | colors             | Canvas, surface, and overlay text colors        |
