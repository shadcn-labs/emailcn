## Usage

```tsx
import { FullWidthTestimonial } from "@/components/emails/marketing/testimonials/full-width-testimonial";

<FullWidthTestimonial variant="overlapping-avatar" />;
```

## API Reference

| Prop      | Type                                | Default        | Description         |
| --------- | ----------------------------------- | -------------- | ------------------- |
| `variant` | `"default" \| "overlapping-avatar"` | `"default"`    | Avatar arrangement  |
| `quote`   | `string`                            | Mailviews copy | Testimonial copy    |
| `theme`   | `EmailThemeTokens`                  | `defaultTheme` | Email theme styling |
