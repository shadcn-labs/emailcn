## API Reference

| Prop      | Type                            | Default        | Description             |
| --------- | ------------------------------- | -------------- | ----------------------- |
| `icon`    | `string`                        | `"✨"`         | Displayed icon          |
| `variant` | `"center" \| "left" \| "right"` | `"center"`     | Inline content position |
| `theme`   | `EmailThemeTokens`              | `defaultTheme` | Email theme styling     |
