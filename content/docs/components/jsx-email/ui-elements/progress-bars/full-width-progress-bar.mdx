## API Reference

| Prop          | Type                        | Default               | Description                           |
| ------------- | --------------------------- | --------------------- | ------------------------------------- |
| `description` | `string`                    | FlowSync description  | Supporting copy used by text variants |
| `theme`       | `EmailTheme`                | `defaultTheme`        | Email theme configuration             |
| `title`       | `string`                    | `FlowSync`            | Progress item title                   |
| `value`       | `number`                    | Variant example value | Progress percentage from 0 to 100     |
| `variant`     | `ProgressBarContentVariant` | `text-top`            | Selects one of the examples above     |
