---
title: "Grid"
description: "Responsive one-to-four-column grid layouts for email content."
---

The six Mailviews grid components share the same markup and content model. This
version keeps one `Grid` component and exposes the column ratios through the
`variant` prop. Text alignment stays independent so it does not multiply the
layout API.

## Usage

```tsx
import { Grid } from "@/components/emails/ui-elements/grids/grid";
```

```tsx
<Grid
  variant="two-columns"
  cells={["Left column", "Right column"]}
/>
<Grid
  variant="one-three-split"
  align="left"
  cells={["Sidebar", "Main content area"]}
/>
```

## API Reference

| Prop      | Type                                                                                                           | Default         | Description                       |
| --------- | -------------------------------------------------------------------------------------------------------------- | --------------- | --------------------------------- |
| `variant` | `"one-column" \| "two-columns" \| "three-columns" \| "four-columns" \| "one-three-split" \| "three-one-split"` | `"two-columns"` | Column arrangement and widths     |
| `cells`   | `string[]`                                                                                                     | `[]`            | Content for each column, in order |
| `align`   | `"left" \| "center" \| "right"`                                                                                | `"center"`      | Text alignment within columns     |
| `theme`   | `EmailThemeTokens`                                                                                             | `defaultTheme`  | Email theme styling               |
