## API Reference

| Prop        | Type                                            | Default        | Description                     |
| ----------- | ----------------------------------------------- | -------------- | ------------------------------- |
| `avatars`   | `AvatarItem[]`                                  | Four avatars   | Images displayed in the group   |
| `size`      | `"xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl"` | `"md"`         | Avatar and overlap size         |
| `plusCount` | `number`                                        | —              | Adds an optional trailing count |
| `theme`     | Email theme type                                | `defaultTheme` | Email framework theme           |
