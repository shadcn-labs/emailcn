---
title: "Container"
description: "Layout container for wrapping email content with responsive padding."
---

One `Container` component covers both Mailviews layouts. Use `mobile="flush"`
for an edge-to-edge mobile container or `mobile="gutters"` for 24px mobile
gutters.

## Usage

```tsx
import { Container } from "@/components/emails/ui-elements/containers/container";
```

```tsx
<Container mobile="gutters">{children}</Container>
<Container mobile="flush">{children}</Container>
```

## API Reference

| Prop       | Type                            | Default        | Description                     |
| ---------- | ------------------------------- | -------------- | ------------------------------- |
| `mobile`   | `"gutters" \| "flush"`          | `"gutters"`    | Side padding behavior on mobile |
| `align`    | `"left" \| "center" \| "right"` | `"center"`     | Horizontal alignment            |
| `maxWidth` | `string`                        | `"600px"`      | Maximum container width         |
| `children` | `React.ReactNode`               | -              | Container content               |
| `content`  | `React.ReactNode`               | -              | Alternate content prop          |
| `theme`    | `EmailThemeTokens`              | `defaultTheme` | Email theme styling             |
