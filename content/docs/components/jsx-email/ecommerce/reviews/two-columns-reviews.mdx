## Installation

<CodeTabs>
<TabsList>
  <TabsTrigger value="cli">Command</TabsTrigger>
  <TabsTrigger value="manual">Manual</TabsTrigger>
</TabsList>
<TabsContent value="cli">

```bash
npx shadcn@latest add @emailcn/two-columns-reviews
```

</TabsContent>
<TabsContent value="manual">

<Steps>
  <Step>Install the following dependencies:</Step>

```bash
npm install jsx-email
```

<Step>Copy the component source into your project.</Step>
  <ComponentSource
    src="registry/bases/jsx-email/ui/ecommerce/reviews/two-columns-reviews.tsx"
    base="jsx-email"
    title="components/email/two-columns-reviews.tsx"
  />
</Steps>

</TabsContent>
</CodeTabs>

## Usage

```tsx
import { TwoColumnsReviews } from "@/components/email/two-columns-reviews";
```

```tsx
<TwoColumnsReviews {...TwoColumnsReviews.PreviewProps} variant="with-divider" />
```
