## Installation

<CodeTabs>
<TabsList>
  <TabsTrigger value="cli">Command</TabsTrigger>
  <TabsTrigger value="manual">Manual</TabsTrigger>
</TabsList>
<TabsContent value="cli">

```bash
npx shadcn@latest add @emailcn/masonry-grid-reviews
```

</TabsContent>
<TabsContent value="manual">

<Steps>
  <Step>Install the following dependencies:</Step>

```bash
npm install jsx-email
```

<Step>Copy the component source into your project.</Step>
  <ComponentSource
    src="registry/bases/jsx-email/ui/ecommerce/reviews/masonry-grid-reviews.tsx"
    base="jsx-email"
    title="components/email/masonry-grid-reviews.tsx"
  />
</Steps>

</TabsContent>
</CodeTabs>

## Usage

```tsx
import { MasonryGridReviews } from "@/components/email/masonry-grid-reviews";
```

```tsx
<MasonryGridReviews
  {...MasonryGridReviews.PreviewProps}
  variant="with-divider"
/>
```
