## Installation

<CodeTabs>
<TabsList>
  <TabsTrigger value="cli">Command</TabsTrigger>
  <TabsTrigger value="manual">Manual</TabsTrigger>
</TabsList>
<TabsContent value="cli">

```bash
npx shadcn@latest add @emailcn/full-width-reviews
```

</TabsContent>
<TabsContent value="manual">

<Steps>
  <Step>Install the following dependencies:</Step>

```bash
npm install jsx-email
```

<Step>Copy the component source into your project.</Step>
  <ComponentSource
    src="registry/bases/jsx-email/ui/ecommerce/reviews/full-width-reviews.tsx"
    base="jsx-email"
    title="components/emails/ecommerce/reviews/full-width-reviews.tsx"
  />
</Steps>

</TabsContent>
</CodeTabs>

## Usage

```tsx
import { FullWidthReviews } from "@/components/emails/ecommerce/reviews/full-width-reviews";
```

```tsx
<FullWidthReviews {...FullWidthReviews.PreviewProps} variant="with-divider" />
```
