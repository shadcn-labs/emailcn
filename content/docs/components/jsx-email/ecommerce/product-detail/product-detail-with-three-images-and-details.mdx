## API Reference

| Prop      | Type                                                                                | Default        | Description              |
| --------- | ----------------------------------------------------------------------------------- | -------------- | ------------------------ |
| `variant` | `"default" \| "rating-top" \| "header-top" \| "rating-aside" \| "rating-aside-top"` | `"default"`    | Mailviews layout variant |
| `theme`   | `EmailThemeTokens`                                                                  | `defaultTheme` | Email theme styling      |
