## API Reference

| Prop      | Type                  | Default        | Description         |
| --------- | --------------------- | -------------- | ------------------- |
| `variant` | `ShoppingCartVariant` | `"basic"`      | layout variant      |
| `items`   | `ShoppingCartItem[]`  | data           | Cart item overrides |
| `theme`   | `EmailThemeTokens`    | `defaultTheme` | Email theme styling |
