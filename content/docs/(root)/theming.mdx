Theming in emailcn is **Tailwind-based**: theme files that export a `TailwindConfig`-compatible object (via `defineEmailTheme` in `registry/themes/define.ts`) that powers `<Tailwind config={theme}>` in React Email and is resolved to concrete values for MJML (via `resolveEmailTheme` in `registry/lib/resolve-theme.ts`).

## Install a theme

Themes are registry items named `theme-<id>`:

```bash
npx shadcn@latest add @emailcn/theme-default
```

Or paste the JSON URL:

```bash
npx shadcn@latest add https://emailcn.com/r/theme-linear.json
```

## Use with React Email

```tsx
import { Tailwind } from "react-email";
import { defaultTheme } from "@/registry/themes/default";

<Tailwind config={defaultTheme}>{/* ... */}</Tailwind>;
```

## Use with MJML

```tsx
import { Tailwind } from "react-email";
import { resolveEmailTheme } from "@/lib/resolve-theme";

const t = resolveEmailTheme(theme);
```
