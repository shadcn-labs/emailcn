## Quick Start

### 1. Install react-email

First, ensure you have react-email installed in your project:

```bash
npm install react-email
# or
pnpm add react-email
```

### 2. Add a Theme

Install a theme to style your emails:

```bash
npx shadcn@latest add https://emailcn.com/r/email-theme-default.json
```

### 3. Add Components

Install individual components:

```bash
# Add a hero component
npx shadcn@latest add https://emailcn.com/r/email-hero.json

# Add a footer component
npx shadcn@latest add https://emailcn.com/r/email-footer.json
```

### 4. Add Blocks

Install complete email templates:

```bash
# Add a welcome email
npx shadcn@latest add https://emailcn.com/r/block-onboarding-default.json

# Add an auth magic link email
npx shadcn@latest add https://emailcn.com/r/block-auth-magic-link-default.json
```

---

## Usage

After installation, import and use components in your emails:

```tsx
import { Body, Container, Head, Html, Preview, Tailwind } from "react-email";

import { Hero } from "@/registry/bases/react-email/ui/hero";
import { Footer } from "@/registry/bases/react-email/ui/footer";
import { defaultTheme } from "@/registry/themes/default";

export function WelcomeEmail() {
  return (
    <Html>
      <Head />
      <Preview>Welcome</Preview>
      <Tailwind config={defaultTheme}>
        <Body className="bg-background font-sans">
          <Container className="mx-auto max-w-container">
            <Hero heading="Welcome!" subheading="Thanks for joining us." />
            <Footer companyName="Acme" />
          </Container>
        </Body>
      </Tailwind>
    </Html>
  );
}
```

## Registry Structure

In this repo, themes and email UI live under **`registry/`**:

- **Themes** — Tailwind configs generated from tokens (`registry/themes/`)
- **Components** — Reusable sections per stack (`registry/bases/react-email/ui/` and `registry/bases/mjml-react/ui/`)
- **Blocks** — Full templates (`registry/bases/react-email/blocks/` and `registry/bases/mjml-react/blocks/`)

All items are compatible with `npx shadcn add` and can be installed individually.
