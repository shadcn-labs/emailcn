## Prerequisites

- **React Email** templates: a project with **`react-email`** installed.
- **MJML** templates: **`@faire/mjml-react`** and **`mjml-browser`** (see [MJML component](/docs/components/mjml-react/hero) pages for the exact `npm` line).

## Registry setup (optional)

Register the namespace once in **`components.json`** (see [Registry](/docs/registry)):

```json
{
  "registries": {
    "@emailcn": "https://emailcn.com/r/{name}.json"
  }
}
```

## Install a theme

Email styles come from **Tailwind email themes** installed under `components/emails/themes/`:

```bash
npx shadcn@latest add @emailcn/theme-default
```

Or use the full URL from any docs page:

```bash
npx shadcn@latest add https://emailcn.com/r/theme-default.json
```

## Install components

**React Email** — component names have **no** prefix:

```bash
npx shadcn@latest add @emailcn/hero
npx shadcn@latest add @emailcn/footer
```

**MJML** — use the **`mjml-`** prefix:

```bash
npx shadcn@latest add @emailcn/mjml-hero
npx shadcn@latest add @emailcn/mjml-footer
```

## Install blocks

Full templates (React Email or MJML) use the `block-` / `mjml-block-` names:

```bash
npx shadcn@latest add @emailcn/block-onboarding-default
npx shadcn@latest add @emailcn/mjml-block-onboarding-default
```

## Usage

After installation, files are written to **`components/emails/`** — themes in **`components/emails/themes/`**, optional font helpers in **`components/emails/fonts/`**.

Each **React Email** UI default export is a **full document** (`Html` → `Head` → `Preview` → `Tailwind` → `Body`). Import **`Section`** exports (e.g. `HeroSection`) when you compose a single custom shell.

```tsx
import { Hero } from "@/components/emails/hero";

export function WelcomeEmail() {
  return <Hero heading="Welcome!" subheading="Thanks for joining us." />;
}
```

Compose with one outer document and section partials:

```tsx
import { Body, Container, Head, Html, Preview, Tailwind } from "react-email";
import { DefaultFonts } from "@/components/emails/fonts/default";
import { FooterSection } from "@/components/emails/footer";
import { HeroSection } from "@/components/emails/hero";
import { defaultTheme } from "@/components/emails/themes/default";

export function WelcomeEmail() {
  return (
    <Html>
      <Head>
        <DefaultFonts />
      </Head>
      <Preview>Welcome</Preview>
      <Tailwind config={defaultTheme}>
        <Body className="bg-background font-sans">
          <Container className="mx-auto max-w-container">
            <HeroSection
              heading="Welcome!"
              subheading="Thanks for joining us."
            />
            <FooterSection companyName="Acme" />
          </Container>
        </Body>
      </Tailwind>
    </Html>
  );
}
```

For **theming**, **font presets**, and **registry URLs**, see [Theming](/docs/theming) and [Registry](/docs/registry).
